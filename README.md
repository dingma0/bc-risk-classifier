# BIOF 520 Assignment 2

Clinical Classifier Challenge: Improving Risk Stratification in TaLG Bladder Cancer

## Files and Directories
`images`: Figures used in the report.

`model`: Final trained model and feature list.

`notebooks`: Python Jupyter notebooks for model training.

`scripts`: R clustering and survival analysis scripts.

`data.zip`: Labelled and unlabelled training/validation data.

`renv.lock`: R lockfile for restoring the R environment.

`requirements.txt`: Python package dependencies.
